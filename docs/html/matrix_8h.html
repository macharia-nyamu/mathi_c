Detailed Description

Mathi C Library - Matrix Utilities

Provides basic matrix operations: addition, subtraction, multiplication, transpose, determinant calculation, identity matrix creation, and scalar multiplication.

Status codes: 0 - SUCCESS 1 - INVALID_INPUT 2 - MATRIX_ERROR 3 - UNKNOWN_ERROR
