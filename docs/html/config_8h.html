Detailed Description

Mathi C Library - Configuration Utilities

This header provides functions for loading, reading, modifying, and saving configuration files (INI-style or similar). It allows retrieval of strings and integers, as well as updating and saving configurations.

Status codes: 0 - SUCCESS 1 - FILE_ERROR 2 - MEMORY_ERROR 3 - KEY_NOT_FOUND 4 - INVALID_INPUT 5 - UNKNOWN_ERROR
