Detailed Description

Mathi C Library - Logging Utilities

This header provides structured logging functions for informational, warning, and error messages. Logs can optionally be written to a specified file.

Status codes: 0 - SUCCESS 1 - FILE_ERROR 2 - INVALID_INPUT 3 - UNKNOWN_ERROR
