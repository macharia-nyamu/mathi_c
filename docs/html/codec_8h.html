Detailed Description

Mathi C Library - Codec Utilities

This header provides encoding, decoding, and compression functions, including Base64 and Run-Length Encoding (RLE).

Status codes: 0 - SUCCESS 1 - MEMORY_ERROR 2 - INVALID_INPUT 3 - UNKNOWN_ERROR
